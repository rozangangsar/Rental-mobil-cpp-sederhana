================================================================================
                        TIPS UNTUK DEBUGGING
================================================================================

Jika ada error saat kompilasi:
1. Pastikan g++ terinstall: g++ --version
2. Lihat file KOMPILASI.txt untuk solusi
3. Verifikasi semua file .cpp ada di folder

Jika ada error saat runtime:
1. Pastikan input.txt ada di folder yang sama
2. Baca pesan error, biasanya jelas apa masalahnya
3. Check file format di input.txt

Jika ingin modifikasi kode:
1. Edit file .cpp atau .h
2. Compile ulang dengan make atau g++ command
3. Run aplikasi lagi
