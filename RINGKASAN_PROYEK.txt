================================================================================
DESKRIPSI SINGKAT
================================================================================
Aplikasi ini adalah sistem manajemen rental mobil yang dibangun menggunakan
C++ dengan pendekatan Object Oriented Programming (OOP). Aplikasi menyediakan
fitur lengkap untuk mengelola kendaraan, penyewa, dan transaksi penyewaan
dengan antarmuka berbasis console.

================================================================================
FITUR UTAMA (15 FITUR FUNGSIONAL)
================================================================================

1. Manajemen Kendaraan:
   - F-01: Tambah Mobil (Register new car)
   - F-02: Edit Mobil (Edit car data)
   - F-03: Hapus Mobil (Delete car data)
   - F-04: Lihat Semua Mobil (View all cars)
   - F-05: Lihat Mobil Tersedia (View available cars)
   - F-06: Cari Mobil Berdasarkan Tipe (Search cars by type)

2. Manajemen Penyewa:
   - F-07: Daftar Penyewa (Register new renter)
   - F-08: Edit Data Penyewa (Edit renter data)
   - F-09: Hapus Data Penyewa (Delete renter data)
   - F-10: Lihat Semua Penyewa (View all renters)

3. Transaksi Penyewaan:
   - F-11: Sewa Mobil (Rent a car)
   - F-12: Kembalikan Mobil (Return a car)
   - F-13: Lihat Riwayat Penyewaan (View rental history)

4. File Management:
   - F-14: Load Data dari File (Load data from input.txt)
   - F-15: Simpan Data ke File (Save data to output.txt)

================================================================================
STRUKTUR CLASS
================================================================================

Class Car
  Attributes: carID, carModel, carType, dailyRate, available, currentRenter
  Methods: Setter/Getter untuk semua attributes + printCarInfo(), printCarStatus()

Class Renter
  Attributes: renterID, renterName, phoneNumber, address, rentalCar,
              rentalDays, totalCost
  Methods: Setter/Getter untuk semua attributes + printRenterInfo(),
           printRentalDetails()

Class RentalManager
  Attributes: cars (vector<Car>), renters (vector<Renter>)
  Methods: Car Management, Renter Management, Rental Transactions,
           File I/O, Search & Filter
